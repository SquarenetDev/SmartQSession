# SmartSession

다중사용자를 위한 사용자세션

Platform : 
iOS / Android 

How to use (Android) :
SmartQUser smartQUser = SmartQUser();
smartQUser.userId = "AAA";
smartQUser.password = "1111";
SmartQSession.setSmartQUser(smartQUser);

How to use (iOS) :
var smartQUser = SmartQSession.SmartQUser()
smartQUser.userId = "AAA"
smartQUser.password = "1111"
SmartQSession.setSmartQUser(smartQUser);
